Pincipais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b>...</b> (não semântica).

Nesta frase, temos um termo em destaque usando a tag <strong>...</strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>...</i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em>...</em> (semântica).

Texto marcado

Podemos criar também um texto marcado udando a tag <mark>...</mark>.
E podemos mudar a cor da marcação usando parâmetro das CSS. <mark style="backgroud-color:cor;">>Texto marcado</mark>.

E neste outro parágrafo, temos outro texto marcado no final. Usando a CSS dentro do documento em <head> para todas a marcas dentro da página.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.
Texto grande: <big>...</big> (tag obsoleta).
Texto pequeno: <small>...</small> (tag ainda operante).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. tag:<ins>...</ins> (semântica).

Existe também o sublinhado com a tag:<u>...</u> (não semântica).
